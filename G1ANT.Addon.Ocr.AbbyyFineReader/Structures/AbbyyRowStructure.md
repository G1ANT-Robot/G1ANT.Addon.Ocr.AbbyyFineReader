# abbyyrow

This structure stores information about a row on a page and has five fields:

| Field   | Type    | Description                                                  |
| ------- | ------- | ------------------------------------------------------------ |
| `cells`  | abbyrow |             |
| `count` | integer |             |
| `top`   | integer |             |
| `bottom` | integer |             |
| `istablerow` | boolean |             |
